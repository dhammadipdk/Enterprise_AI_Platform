# Vehicle Behavior Contextual Intelligence

Version: 1.0.0

Status: Active

# Purpose

The Vehicle Behavior domain models how a vehicle is used, how it ages, how its mechanical systems perform, and how operational characteristics influence insurance risk, maintenance requirements, claim probability, fraud detection, and personalized recommendations.

Unlike Environment, Infrastructure, and Mobility, which describe external conditions, Vehicle Behavior captures the intrinsic operational state and usage profile of the insured vehicle.

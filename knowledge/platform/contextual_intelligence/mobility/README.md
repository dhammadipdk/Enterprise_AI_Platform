# Mobility Contextual Intelligence

Version: 1.0.0

Status: Active

---

# Purpose

The Mobility domain models how vehicles move across the transportation network and how mobility patterns influence accident probability, vehicle exposure, claim frequency, travel efficiency, and insurance risk.

Unlike the Infrastructure domain, which focuses on static assets, Mobility focuses on dynamic transportation behavior.

---

# Enterprise AI Memory

Version : 1.0

Status : Active

---

## Purpose

The Memory subsystem provides persistent, contextual and operational memory for all AI agents within InsureAI.

Unlike the Knowledge Repository, which stores enterprise facts, Memory stores information required for reasoning, planning, execution and personalization.

Memory combines semantic knowledge, contextual observations, graph structures, embeddings, reasoning traces and conversation history into a unified enterprise memory architecture.

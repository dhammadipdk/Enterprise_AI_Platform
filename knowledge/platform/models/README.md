# Enterprise AI Model Registry

Version : 1.0

Status : Active

---

## Purpose

The Model Registry provides centralized governance for all AI models used across the Enterprise AI Platform.

It manages model discovery, deployment, lifecycle, inference, monitoring, versioning and operational policies while remaining provider independent.

The registry supports foundation models, embedding models, rerankers, multimodal models, domain models and fine-tuned enterprise models.
